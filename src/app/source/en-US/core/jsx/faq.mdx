## Common Questions

### What is the difference between jsx-runtime and babel-plugin-jsx?

jsx-runtime is designed to work with compilers that support `pragma` configuration, all transforms use standard jsx syntax. babel-plugin-jsx is the official Gyron babel plugin with more features than the jsx-runtime runtime.

### Why isn't state updating?

Please check if the setup option is enabled in your bundler config. Set it to true and try again.
