## Hello World

此页面是展示如何使用 Gyron JSX 语法创建一个 Hello World 节点，以及最终节点的内容。

```jsx
const App = <h1>Hello World</h1>
// ↓ ↓ ↓ ↓ ↓
import { jsx } from '@gyron/jsx-runtime'
const App = jsx('h1', {
  children: 'Hello World',
})
```

上述的转换过程是通过其它平台的能力，比如 babel。在这里[JSX](/core/jsx/getting-started)可以看到代码转换的具体信息。

如果你需要知道项目如何初始化，可以来这里[（初始化）](/docs/tutorial#创建一个新的%20Gyron%20应用)阅读更多信息。
